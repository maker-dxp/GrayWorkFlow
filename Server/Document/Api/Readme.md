 | code | message |
 | ---- | ---- |
 | 200  |   OK    |

 | 4000 | Body为空 |

 | 4011 | 非法Token |
 | 4012 | Token已过期 |
 | 403 | 禁止访问 |
 | 405 | 方法不允许 |
 | 409 | 存在冲突 |
 | 4090 | 用户已存在 |
